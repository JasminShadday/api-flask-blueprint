## Cria o seu ambiente virtual

```php
https://flask.palletsprojects.com/en/3.0.x/installation/
```

## Execute os seguintes comando dentro do ambiente virtual

```php
pip install flask
pip install mysql-connector-python
pip install flask_cors
```
## Execute a Aplicação

```php
python3 app.py 
```
